Projects

Mail Drome - An universal mail application system

The project “Mail Drome” is a web application in which a user can access multiple mail servers at single place. The entire development of the application was done using JSP, servlets, Java mail technology and oracle database. The project is all about creating a web application that works as a mail system with an added feature of including many mail servers within a single page. It includes all the functions involved with a mail application and includes many mail servers. A universal mail application can be useful to the user when he has to manage many accounts in different mail servers and when he wishes to have a universal account that opens up all the mail directories at one time The project is yet to be extended in many perspectives. Because of time constraint the main idea only was developed.
